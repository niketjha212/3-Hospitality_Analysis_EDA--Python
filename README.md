# AtliQ Grand Hospitality Analytics Project 

## Overview:

This project focuses on ad-hoc data analysis in the hospitality domain for AtliQ Grand.

The objective is to generate actionable insights into:

- Hotel operations
- Room bookings
- Revenue performance
- Guest behavior

These insights are designed to support data-driven decision-making and improve business efficiency.
The analysis covers hotel types, room categories, occupancy trends, revenue realization, and customer ratings.

## Project Scope:
- Analyze hotel performance across different locations and hotel types
- Study room category occupancy rates and revenue realization
- Identify trends in customer ratings and booking patterns
- Provide visual summaries through clear and intuitive visualizations

## Tools & Technologies:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
